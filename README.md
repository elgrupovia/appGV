# App Grupo Via

Esta aplicación es desarrollada para **Grupo Via** y está compuesta por dos partes principales:

- **Frontend:** Flutter
- **Backend:** Laravel

## Descripción General

La app permite a los usuarios interactuar con los servicios y datos de Grupo Via a través de una interfaz moderna y multiplataforma (iOS, Android, Web, etc.), utilizando Flutter para el frontend y Laravel para el backend.

## Integración con API Grupo Via

Todas las funcionalidades principales de la app se comunican con la **API de Grupo Via** para obtener y enviar información relevante. Esta API es el núcleo de la integración entre la app y los servicios internos de Grupo Via.

## Web Corporativa

La web oficial de Grupo Via es: [https://grupovia.net/](https://grupovia.net/)

Esta web está conectada al sistema Zoho y utiliza WordPress como gestor de contenidos. La integración permite sincronizar información relevante entre la web y los sistemas internos de Grupo Via.

## Tecnologías Utilizadas

- **Frontend:** [Flutter](https://flutter.dev/)
- **Backend:** [Laravel](https://laravel.com/)
- **API:** API propia de Grupo Via
- **Web:** WordPress + Zoho

## Estructura del Proyecto

- `frontend/`: Código fuente de la app Flutter
- `backend/`: Código fuente del backend Laravel
